# Monitoraggio dell’Inquinamento — Air Forecast

Questo repository pubblica una dashboard **Looker Studio** tramite **GitHub Pages** e include i file principali del progetto:
- dataset (`.xlsx`)
- script di elaborazione/previsione (`.py`)
- documentazione (`.docx`)
- report (`.pdf`)

## 🔗 Dashboard
La dashboard è visibile qui (GitHub Pages):
- https://<TUO-UTENTE>.github.io/<NOME-REPO>/

La dashboard è incorporata da Looker Studio tramite iframe.

## 📁 Struttura repository
- `index.html` → pagina web che incorpora la dashboard Looker Studio
- `files/` → materiali del progetto:
  - `new meteo e air.xlsx`
  - `air_forecast.py`
  - `AIR_FORECAST.docx`
  - `MONITORAGGIO_DELL'_INQUINAMENTO (1).pdf`

## 🚀 Pubblicazione (GitHub Pages)
1. Carica `index.html` nella root del repository
2. Carica i file nella cartella `files/`
3. Vai su **Settings → Pages**
4. Seleziona **Deploy from a branch** → Branch `main` → Folder `/ (root)`
5. Salva: GitHub ti fornirà l’URL pubblico

## 🔐 Permessi Looker Studio (importante)
Per vedere la dashboard incorporata:
- il report Looker Studio deve essere condiviso con **“Chiunque abbia il link”** (visualizzatore)
- deve essere abilitato **Embed report**
